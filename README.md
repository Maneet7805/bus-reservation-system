# bus-reservation-system
A C-based bus ticket reservation system with user authentication, booking, and admin management of buses and schedules. Data is stored locally using file handling.

## Technologies
- C Programming
- File Handling

## Features
- User registration and login
- Book and cancel tickets
- Admin panel to manage buses and schedules and create deatiled reports
- Save data in local files

## How to Run
1. Open terminal in project folder.
2. Compile the program:
```bash
gcc main.c -o bus_reservation
